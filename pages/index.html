Core Intelligence

Self-Editing Core

Xenon doesn't just talk. It acts. Using PyO3 and an autonomous logic loop, it creates, tests, and saves global functions to its own framework in real-time.

4-bit Context

Optimized for performance. Context is stored in high-density 4-bit Protobuf structures and translated back to 16-bit only when required by the inference engine.

Tauri 2.0 Architecture

Native performance with a minimalist footprint. Xenon leverages Rust for high-speed backend tasks while providing a beautiful, hardware-accelerated UI.

Multi-Provider

Full support for OpenAI, Anthropic, Gemini, DeepSeek, and local GGUF models via llama-server integration. Your agent, your choice of brain.

Built for the Future

Xenon includes native integrations for system-level operations. From headless browser control via Playwright to direct PowerShell execution, Xenon has the keys to your workspace.

  • Embedded Python (3.13 & 3.10) with automated pip management.
  • Browser data import (Chrome, Edge, Firefox) for seamless context migration.
  • Desktop shortcut and TUI-based installation.
